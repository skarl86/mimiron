# Mimiron

Claude Code용 적응적 멀티 에이전트 하네스. clarify → spec → plan → execute → evaluate → finalize의 6단계를 *정량 게이트* 기반 전이로 운영. DAG 병렬 실행과 안전핀이 박힌 persistent 루프가 핵심.

> 상태: Phase A 자동화 작업 완료 (Phase 0, A1, A2, A3, A4, A5, A6.1–A6.6, B0). 사용자 직접 작업 남음: A6.7 (B01 벤치마크 큐레이션), A6.8 (B01 수동 종주). Phase B 진입은 `docs/ralph-loop-entry.md` 참고.
